# Home Automation for Autistism.
### 

This application was created to schedule Autistic activities by home automation.

app write in ES6 with:
- node.js
- express.js
- socket.io
- bootstrap
- mongodb
- node-red
- siemens s7
- siemens opcua
- alexa
- eWeLink sonoff MINIR2
- Hue Philips
- home assistant
- google calendar

node-red is included in express application.
